## Project Overview

BF591 final project: an R Shiny app exploring post-mortem Huntington's Disease RNA-seq data (GSE64810). 69 samples total — 20 HD, 49 neurologically normal controls — from human BA9 prefrontal cortex. All four tabs are complete and in `app/app.R`. All preprocessing is complete.

## Data Pipeline Commands

Run from project root in RStudio terminal or PowerShell:

```r
# Full pipeline (download → process → validate)
Rscript scripts/run_all_data_prep.R

# Individual steps
Rscript scripts/01_download_data.R        # downloads 3 .gz files into data/raw/
Rscript scripts/02_prepare_processed_csvs.R  # ~2 min; writes 5 CSVs to data/processed/
Rscript scripts/03_validate_processed_csvs.R # sanity checks all processed files
```

## Processed CSV Schemas

### `sample_info.csv`
Key columns: `sample_id` (C_#### or H_####), `geo_accession`, `diagnosis` (Control|HD), `age_at_death`, `pmi_hours`, `rin`, `vonsattel_grade`, `cag_repeat`, `hv_striatal_score`, `hv_cortical_score`. `sex` is absent from GEO series matrix — column exists but is all NA.

### `counts_wide.csv`
Columns 1–3: `gene_id` (Ensembl), `gene_symbol`, `gene_name`. Columns 4–72: one numeric column per sample (sample IDs match `sample_info$sample_id`). To extract the numeric matrix in R: `counts_wide[, -(1:3)]`.

### `differential_expression.csv`
Standard DESeq2 columns plus: `neg_log10_padj`, `significance` (FDR<0.05 | Not significant), `direction` (Up in HD | Down in HD | No significant change), `rank` (sorted by padj asc, then |log2FC| desc). Comparison is HD vs. Control.

### `gene_expression_long.csv`
Long-format pivot of counts_wide joined to sample_info. Columns: `gene_id`, `gene_symbol`, `sample_id`, `diagnosis`, `normalized_count`, `log2_normalized_count`, `age_at_death`, `sex`, `vonsattel_grade`, `cag_repeat`. **147 MB — do not load this file in the Shiny app**; pivot from counts_wide + sample_info reactively instead.

## Planned Shiny App Architecture

The app goes in `app/app.R`. Use `navbarPage` at the top level with nested `tabsetPanel` per tab. Each tab has its own `fileInput` and `sidebarLayout`.

### Tab 1 — Sample Information Exploration
Input: `sample_info.csv`. Three sub-tabs: summary table (column name / type / mean±sd or distinct values), sortable `DT::datatable`, and distribution plots (histogram/violin of a chosen continuous column grouped by a chosen categorical column).

### Tab 2 — Counts Matrix Exploration
Input: `counts_wide.csv`. Sliders: minimum variance percentile, minimum non-zero samples. Four sub-tabs: filter summary text, diagnostic scatter plots (median vs. variance; median vs. zeros, colored by pass/fail filter), clustered heatmap (`pheatmap`, with log-transform toggle), PCA scatter (user picks PC axes; label with % variance explained). Compute everything from the wide matrix — skip columns 1–3.

### Tab 3 — Differential Expression
Input: `differential_expression.csv`. Two sub-tabs: sortable `DT::datatable` with gene search, volcano plot (Assignment 7 style — x/y axis picker, color pickers, padj slider).

### Tab 4 — Individual Gene Expression Visualization
Inputs: `counts_wide.csv` + `sample_info.csv`. User picks a gene (searchable selectize), a grouping column, and a plot type (boxplot/violin/bar/beeswarm). Plot renders on button click. Table shows the joined long-format counts for the selected gene.

## Key Implementation Notes

- Each tab's file load is its own isolated `reactive({})` — tabs do not share loaded data.
- PCA: `prcomp()` on the transposed filtered numeric matrix (samples as rows).
- Heatmap: `pheatmap` package. Log-transform with `log2(x + 1)` before passing to pheatmap.
- Required packages: `shiny`, `bslib`, `ggplot2`, `DT`, `pheatmap`, `colourpicker`, `tidyr`, `RColorBrewer`, `ggbeeswarm`.
- The example volcano plot in the project (Assignment 7 code by T. Falk) uses `ggplot2` with `.data[[col_name]]` for column selection — keep this pattern.

## Dataset Context

- DE comparison: HD vs. Control (HD is the numerator for log2FoldChange).
- Counts are DESeq2 size-factor normalized, not raw integer counts — do not re-run DESeq2 normalization.
- `vonsattel_grade`, `cag_repeat`, `age_of_onset`, `disease_duration_years` are NA for all Control samples (HD-specific covariates).
- `sex` is missing for all samples in this GEO series matrix extract.

## Assignment Rubric (BF591 Final Project)

### General requirements (apply to all tabs)
- Basic CSV validation: check file is well-formatted, show an error message otherwise.
- All buttons must have labels; all sections need text descriptions explaining what each input does.
- Write comments for non-obvious Shiny commands or logic.

### Tab 1 — Sample Information (required)
- Sub-tab: summary showing number of rows, number of columns, and a table with column name / type / mean±sd (for numeric) or distinct values (for categorical).
- Sub-tab: sortable data table of the full sample info.
- Sub-tab: histograms, density plots, or violin plots of continuous variables. User can choose which column to plot and which column to group by.

### Tab 2 — Counts Matrix Exploration (required)
- Sliders: (1) include genes at ≥ X percentile of variance, (2) include genes with ≥ X non-zero samples.
- Sub-tab: summary text — number of samples, total genes, genes passing/failing filter (count and %).
- Sub-tab: two scatter plots (median count vs. variance; median count vs. zeros) — passing genes darker, filtered genes lighter.
- Sub-tab: clustered heatmap of filtered genes, with log-transform toggle and color legend.
- Sub-tab: PCA scatter — user selects which PCs to plot; axis labels include % variance explained.

### Tab 3 — Differential Expression (required)
- Sub-tab: sortable table; optional gene name search to filter rows.
- Sub-tab: volcano plot matching Assignment 6/7 style (x/y axis selector, color pickers, padj magnitude slider).

### Tab 4 — Individual Gene Expression (chosen)
Two file inputs (counts_wide.csv, sample_info.csv), searchable gene selectize, group-by selector, plot-type radio buttons, action button. Plot is button-bound (`bindEvent`). Table below updates reactively. Pure helpers: `get_gene_counts`, `pivot_gene_to_long`, `make_gene_plot`.
